# Matem-tica-com-python
Este projeto consiste na implementação de modelos matemáticos avançados (lineares, exponenciais, logarítmicos, quadráticos e racionais) para resolver desafios reais de engenharia e desenvolvimento de sistemas. Através de uma abordagem que une resolução analítica e implementação em Python. O foco é transformar dados teóricos em decisões técnicas.
